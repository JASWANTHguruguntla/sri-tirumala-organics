# 🌿 Sri Tirumala Organic Oils & Millets

A responsive single-page e-commerce website for **Sri Tirumala Organic Oils & Millets** — a traditional cold-pressed organic oil and millet business from Rayalaseema, Andhra Pradesh, India.

![Made with HTML/CSS/JS](https://img.shields.io/badge/Built%20with-HTML%20%7C%20CSS%20%7C%20JS-green)
![License](https://img.shields.io/badge/License-All%20Rights%20Reserved-gold)

## ✨ Features

- 🛒 **Product Catalog** — 7 organic oils + 5 organic millets with images, prices & stock status
- 💬 **WhatsApp Ordering** — One-click order via pre-filled WhatsApp messages
- 📱 **Mobile-First Responsive** — Works beautifully on all devices
- 🎨 **Traditional Indian Design** — Warm, earthy, green-and-gold aesthetic
- 🌐 **Bilingual** — English + Telugu (తెలుగు) support
- 🛍️ **Cart System** — Add products, adjust quantities, live running total
- 📋 **Product Detail Modals** — Full descriptions and health benefits
- ✨ **Scroll Animations** — Smooth reveal effects on scroll
- 🔔 **Notify Me** — Out-of-stock items redirect to WhatsApp notification request

## 🚀 Quick Start

Simply open `index.html` in a browser, or serve with any static file server:

```bash
# Using Python
python -m http.server 8080

# Using Node.js
npx serve .
```

Then open [http://localhost:8080](http://localhost:8080)
